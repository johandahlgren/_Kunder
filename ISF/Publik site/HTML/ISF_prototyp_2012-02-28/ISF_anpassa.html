Anpassa webbplatsen

Du kan anpassa hur webbplatsen visas genom att ändra kontrast och storlek.

Ändra storlek

Håll ner Ctrl-tangenten samtidigt som du scrollar upp eller ner med scrollhjulet på musen eller håll ner Ctrl samtidigt som du trycker på plus- eller minus-tangenten (upprepade gånger för gradvis mer effekt) för att förstora eller förminska hela webbplatsen i webbläsarfönstret.

Du kan också ändra textstorlek i webbläsarens menyer, men då kommer t.ex. bakgrundbilder och knappar inte att skalas korrekt.

Ändra kontrast

Använd formuläret nedan för att byta mellan standardläge (mörk text på ljus bakgrund) och högkontrastläge (ljus text på mörk bakgrund).

För att din webbläsare skall kunna spara inställningarna för kontrast så måste webbläsaren acceptera så kallade cookies.
